#####  Auto Sales Inventory System

##### 

##### A Java Swing GUI desktop application for managing auto sales 

##### inventory at a car dealership.

##### 

##### \## Features

##### \- Add new vehicles to inventory with full details

##### \- Search for vehicles by car model

##### \- Displays inventory in a scrollable table

##### \- Nigerian Naira (₦) currency formatting

##### \- Date picker for date added

##### \- Car status tracking (Available / Sold)

##### \- Custom table renderer for multi-line descriptions

##### \- Dropdown menus for Make, Color, Transmission, Fuel and Engine type

##### 

##### \## Technologies Used

##### \- Java

##### \- Java Swing (GUI)

##### \- Java Collections (HashMap)

##### \- JDateChooser (JCalendar library)

##### \- NetBeans IDE

##### 

##### \## How to Run

##### \- Download and install Java on your computer

##### \- Open the project in any Java IDE (NetBeans, IntelliJ, Eclipse)


##### \- Run the AutoSalesMainClass.java file
